Railway OpenTelemetry Demo App

Welcome to the OpenTelemetry demo app running in Railway

About this demo

This application complements the OpenTelemetry Collector + Backend Template found in the Railway template marketplace.

If you'd like more information on how to implement OpenTelemetry in your applications in Railway, check out the OpenTelemetry Tutorial in Railway's documentation.

How to interact with this app

1. Generate some traffic to the /rolldice route (update the number of rolls for more fun): /rolldice?rolls=10

2. Check out the telemetry data you just generated in the backend services:

  • Jaeger
  • Zipkin
